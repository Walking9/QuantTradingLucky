# Reports

研究报告、因子评估、策略回测结果的归档目录。

## 子目录

```
reports/
├── factors/           # 单因子 / 多因子研究报告
├── strategies/        # 策略研究报告
├── risk/              # 组合风险 / 压力测试报告
├── figures/           # 共享图表资源
└── monthly/           # 每月学习产出的综合报告
```

## 产物格式

- **主报告**：Markdown（易 diff、易评审）。
- **交互图表**：Plotly HTML（保留交互性）。
- **最终快照**：导出 PDF 供外部分享。

`.html` 与 `.pdf` 已在 `.gitignore` 中忽略；请把**源 Markdown**纳入版本控制，导出产物视情况决定。

## 最小报告模板

每份报告建议包含以下章节：

1. **摘要** — 2-3 句话描述结论。
2. **假设与数据** — 区间、标的池、数据源、清洗方式。
3. **方法** — 公式 / 伪代码 / 关键实现细节。
4. **结果** — 核心指标表 + 可视化。
5. **稳健性** — 参数敏感性、子区间、子池。
6. **局限性** — 这个结论在什么情况下会失效？
7. **下一步** — 下一轮迭代的假设。
