## 3. 学到的关键概念

1. **复权因子的重要性**：A 股存在频繁的分红送转，必须统一使用前复权（或原始价+复权因子），否则在回测中会出现巨大的虚假收益或回撤。
2. **向量化思维**：在 Pandas/NumPy 中，尽量避免使用 `for` 循环遍历 DataFrame 的每一行。使用内置的向量化运算或 `apply`，能让数据处理速度提升几个数量级。
3. **Parquet 格式的优势**：相比 CSV，Parquet 作为列式存储，不仅压缩率高，且在读取特定列（如仅读取收盘价）时 I/O 极快，非常适合量化时间序列数据。
4. **不同市场的数据对齐**：美股、A股的交易日不同，加密货币是 24/7 交易。在做跨市场相关性分析时，必须先进行正确的时间戳对齐（如 `reindex` 或 `merge_asof`），并处理缺失值（如前向填充 `ffill`）。

## 4. 踩过的坑

对应 `docs/pitfalls.md` 中的记录：

- 【新增】**A股复权处理错误**：起初直接下载未复权数据计算收益率，导致除权日出现动辄 -30% 的“暴跌”。修正方案：统一采用前复权数据进行 EDA，后续回测引擎再考虑原始价+复权因子的动态处理。
- 【新增】**时区混淆**：加密货币通常是 UTC 时间，而 A股/美股是本地时间。合并数据时引发了 pandas 的 `TZ-naive` 和 `TZ-aware` 冲突。修正方案：将所有时间序列统一转换为 UTC 存储，显示时再转为本地时间。
