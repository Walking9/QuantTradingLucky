# 踩坑实录 (Pitfalls Journal)

> 每一个踩过的坑都值千金。**遇到问题 → 定位 → 修复 → 记录**。
> 记录格式越简单越好，保证会持续写下去。

## 使用说明

- 每次解决一个非平凡的 bug 或发现一个思维陷阱，添加一条记录。
- 分类参考《学习计划》第 6 节，但新类别也欢迎。
- 尽量带上：**现象、根因、修复、教训**四要素。
- 严重性分级：🔴 Critical / 🟠 Major / 🟡 Minor。
